 
  1 What is the difference between var, let, and const?
  2 What is the spread operator (...)?
  3 What is the difference between map(), filter(), and forEach()?
  4 What is an arrow function?
  5 What are template literals?


1.var is function scoped. It can be redeclared and reassigned.
let is block scoped. It can be reassigned but cannot be redeclared.
const is block scoped. It cannot be reassigned and cannot be redeclared.

2.Spread Operator used to expand or spread elements of an array or object into individual elements.

3.map transforms each element and returns a new array.
filter returns a new array with elements that match a condition.
forEach executes a function for each element and returns nothing.

4.An arrow function is a shorter way to write functions in JavaScript using the => syntax. It makes code clean and shorter.

5.Template literals are strings written with backticks (`) that allow embedding variables and expressions using ${} and support multi-line strings.
